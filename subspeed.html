Abstract

Program to linearly change timestamps in a subtitle (*.srt) file. It is meant to offset the "PAL acceleration" of videos.

Usage

subspeed.pl infile [+|-]

The program doesn't change the actual file: It reads the input file and writes the output to <STDOUT>.

example

If you have a video file coming from a PAL source and a subtitle file indexed for a NTSC or a digital source, the subtitles will lag about half a second per minute. In this case, you should use the "-" parameter:

subspeed.pl film.srt - > film.new.srt

note

This program will not correct fixed offsets, like for longer and shorter title sequences. Most media players already offer the possibility to set such a fixed subtitle offset.

installation

On *nix style operation systems, install the program with

sudo cp subspeed.pl /usr/local/bin; sudo chmod a+x /usr/local/bin/subspeed.pl
